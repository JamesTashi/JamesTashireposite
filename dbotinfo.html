About

The purpose of the bot:

I premade a Discord bot for my Discord hangout, which performs moderation and other tools. It will be written with node.js. So far we got some components in like erasing messages, kicking and banning members. This project will not be open-source at the moment but we can take ideas if you wish. Log in and add him right now!

The permissions needed to access the bot:

  • Manage Server
  • Manage Roles
  • Manage Channels
  • Kick Members
  • Ban Members
  • Manage and Change Nicknames
  • Read and Manage Messages
